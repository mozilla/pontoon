# Renaming a Project

Renaming a project changes its **Name** and, if desired, its **Slug** (used in URLs).

## Steps

1. Access Pontoon's admin console → the project you want to rename.
2. Update the **Name** field.
3. If you also want to update the slug, update the **Slug** field. Be aware that this changes all URLs for the project and localization pages.
4. Click **SAVE PROJECT**.

!!! warning
    Changing the **Slug** invalidates any bookmarked or shared links to the project's pages. Coordinate with the team and update any external references (documentation, bug trackers, etc.) before or immediately after renaming.

!!! note
    For VCS-backed projects, renaming does not affect the repository or its file structure.
